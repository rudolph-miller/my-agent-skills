以下の次アクション提案書をレビューしてください。

観点:
- 提案の妥当性と優先度の適切さ
- 既存 PRD / Todo との整合性
- 見落としている課題やリスク
- 実現可能性と工数の妥当性
- アクション間の依存関係や実施順序の問題

Red Team 観点（批判的レビュー）:
- 前提条件への疑義: この提案が成り立つための前提は正しいか
- 楽観的見積もりの検出: 優先度・期待効果・スコープが甘くないか
- 代替アプローチ: 別の順序や別のアクションの方が効果的ではないか
- 失敗シナリオ: この計画通りに進めた場合に起きうる最悪のケース
